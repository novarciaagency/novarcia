2. Datos que se pueden recoger

A través del formulario de diagnóstico o contacto se pueden solicitar datos como nombre, empresa, ciudad, email, teléfono, web o redes sociales del negocio, información sobre objetivos comerciales y presupuesto publicitario aproximado.

3. Finalidad del tratamiento

Los datos se tratarán para responder a solicitudes de información, valorar si Novarcia Agency puede ayudar al negocio solicitante, proponer una llamada de diagnóstico y mantener comunicaciones relacionadas con los servicios solicitados.

4. Base legal

La base legal del tratamiento es el consentimiento del usuario al enviar el formulario y, en su caso, la aplicación de medidas precontractuales solicitadas por el interesado.

5. Conservación de los datos

Los datos se conservarán durante el tiempo necesario para atender la solicitud y realizar seguimiento comercial razonable. Si no existe relación comercial, se eliminarán cuando dejen de ser necesarios o cuando el interesado solicite su supresión.

6. Destinatarios

No se venderán datos personales a terceros. Para gestionar formularios, correo, alojamiento web o herramientas de trabajo se pueden utilizar proveedores tecnológicos como Microsoft, GitHub u otros servicios necesarios para el funcionamiento de la actividad.

7. Derechos de los usuarios

El usuario puede solicitar el acceso, rectificación, supresión, oposición, limitación y portabilidad de sus datos escribiendo a novarciaagency@outlook.com. También puede presentar una reclamación ante la Agencia Española de Protección de Datos si considera que el tratamiento no se ajusta a la normativa.

8. Seguridad

Novarcia Agency aplicará medidas razonables para proteger los datos personales frente a accesos no autorizados, pérdida o uso indebido.

9. Cambios en esta política

Esta política podrá actualizarse para adaptarse a cambios legales, técnicos o de actividad. La versión vigente será la publicada en esta página.
