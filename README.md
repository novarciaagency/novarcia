# Novarcia Agency Website

Web estática para GitHub Pages.

## Archivos
- `index.html`: página principal
- `styles.css`: estilos
- `aviso-legal.html`: aviso legal
- `privacidad.html`: política de privacidad
- `cookies.html`: política de cookies
- `assets/`: imágenes de marca

## Subir a GitHub
1. Crear repositorio público `novarcia`.
2. Subir todos los archivos y carpetas de este directorio.
3. Activar GitHub Pages en Settings > Pages > Deploy from branch > main > root.
4. Añadir dominio personalizado `novarcia.com` cuando esté listo.

## Pendiente legal
Completar los campos entre corchetes en aviso legal y privacidad antes de usar la web comercialmente.
